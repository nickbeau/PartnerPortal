Operation

 

The SharePoint Document Migrator consists of a single Window. Fields are as follows:

 

•SharePoint Site URL: Enter here the full path to the site you are uploading files to. For example, to upload files to https://hubone.sharepoint.com/documentcenter, you must enter the whole path, not a subset.

•SharePoint Target Root Folder: Enter here the root folder you are uploading to, for example Documents for the Documents document library

•Your Document will go to: Shows where the document will be uploaded to

•SharePoint User: An email address with rights to upload files to the target location

•Password: The password of the user given above

•Email for Notifications: Who you would like to receive notifications of completion

•Client Source Directory: The folder you are uploading from

•Log FIles Location: Which folder to store log files in

•Maintain Files Dates: Maintains file dates locally and in SharePoint

•Overwrite All Files: Overwrites all files whether newer or not

•Disable Threading: The migration manager will use 25 threads if possible. If needed you can disable threading which will dramatically slow the process down.

 

 

When ready to go, click Process
