Remote Desktop Protocol for Macs

Remote Desktop Protocol for Macs 

 

This process only needs to be completed once per machine.

Navigate to https://itunes.apple.com/us/app/microsoft-remote-desktop/id714464092?mt=8 or search Microsoft Remote Desktop in the AppStore and click Install.

 

When finished installing, click Open.

Go to http://microsoftcie.com.

Navigate to the RDP page for your session and click Connect on your preferred persona.

 

Go to Downloads and open the RDP connection that was downloaded.

 

Click Continue on the Verify Certificate dialog box.

 

On the login screen, select Other User in the bottom left-hand of the screen.

 

The user name will be automatically inserted. Next, enter the password: pass@word1.

 

Press Enter to log in.
