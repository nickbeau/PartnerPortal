Understanding the Platform

Purpose

The Kairos Immersion Experience (KIE) platform provides a solution for those in the field who need to set up and configure a clean and fully configured, lab-like environment that will be used to engage their customers in an interactive manner.

 

KIE can be only be delivered using Office 365 services.

 

This Technical Guide contains critical instructions that must be followed in order to set up and configure the environment correctly prior to your first and all subsequent sessions. It provides detailed information regarding the hardware, software, network and accounts/passwords that are a preface to delivery of your first KIE session.

 

It should be noted that the KIE platform is built on the Microsoft CIE Platform and that all commentary for CIE is required for KIE, not vice versa.

 

Setup Time

When using KIE for Office 365 platform deployments, you can obtain a QuickTenant immediately at www.MicrosoftCIE.com or create a tenant with a custom domain name. If you need a custom domain name, it is recommended that you start the Office 365 provisioning process a week in advance of the initial setup of the remainder of the KIE environment. All setup should be scheduled and completed well in advance of your first KIE session.

 

Partners wishing to perform a KIE need to be certified at www.microsoftcie.com before they commence any of the procedures in this document. 

 

Other Considerations

For technical support on the CIE Platform, contact support@microsoftcie.com. CIE technical support has a one-business-day Service Level Agreement (SLA) based on Central Time (CT). Please keep this in mind when planning setup, should you encounter any issues.

 

For technical support on the KIE Platform additions, contact support@hubone.com. KIE technical support has a one-business-day Service Level Agreement (SLA) based on Australian Eastern Standard Time (AEST). Please keep this in mind when planning setup, should you encounter any issues.
