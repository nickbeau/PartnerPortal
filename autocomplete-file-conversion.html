Autocomplete File Conversion

xample of information needed:

 

Old AutoComplete File: Steam_Autocomplete_a123b234 (Obtained from Old System)

 

New AutoComplete File: Steam_Autocomplete_c345d456 (Current System/Configuration)

 

Location of AutoComplete File: C:\users\<username>\appdata\local\microsoft\outlook\roamcache

 

--------------------------------

 

In the old System navigate to C:\users\<username>\appdata\local\microsoft\outlook\roamcache

 

Copy the Steam_Autocomplete_a123b234 (example name) file to a location where you can transport it to the new system.

 

After you have copied the file to an accessible location go back to your new system.

 

ensure outlook is closed

 

navigate to C:\users\<username>\appdata\local\microsoft\outlook\roamcache

 

rename Steam_Autocomplete_c345d456 (example name) to Steam_AutocompleteOld_c345d456

 

copy Steam_Autocomplete_a123b234 into C:\users\<username>\appdata\local\microsoft\outlook\roamcache

 

rename Steam_Autocomplete_a123b234 to Steam_Autocomplete_c345d456

 

Open Outlook and you will see that the AutoComplete that you have copied accross will be in affect.
