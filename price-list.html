HubOne Services are updated regularly and pricing is available via your distributor. A detailed description of our products and services follow:

 

Mail Manager

Mail Manager allows customers to automatically save emails to the OnePractice SharePoint library, add meta data and extract attachments. Emails are saved in .msg format meaning they can be searched and immediately opened in outlook for reply and more. Requires Office 365 E and the OnePractice Portal. Mail Manager also allows the creation of emails and quick attachment of files from SharePoint Online to new emails. Mail Manager runs in Outlook or Outlook Web App

Office 365 in 365

365 in 365 is a training program offering a combination of web-casts, videos, emails and other mechanisms to assist a team get up to speed in Office 365 and the HubOne suite.

OnePractice Portal

OnePractice Portal includes the relevant SharePoint sites, additions, tables and automations to enable the use of Mail Manager, Template Manager, Scanned Document Management and more. Requires SharePoint Online Enterprise SKUs at a minimum.

OnePractice Suite

The OnePractice Suite includes Mail Manager, the OnePractice Portal, Template Manager, Scanned Document Manager, Migration Manager and HubOne Support.

OnePractice Support

OnePractice Support is support of the above products via telephone, web or email.

OnePractice Scanned Document Manager

Scanned Document Manager reads searchable PDF files and automatically renames and uploads the file to the appropriate client folder, where SharePoint automation and workflow can drive emails and approval rules. Scanned Document Manager runs on PC Operating Systems and will watch a folder for rapid integration with modern scanners and their software.

Template Manager

Template Manager allows for rapid creation of documents and spreadsheets incorporating data from the user (prompted), Client, Contact and Job information from the practice management system. Template Manager allows for easy creation of templates, with a rapid field builder and does not require office to create, populate and save documents.

OnePractice Migration Manager

OnePractice Migration Manager is an easy migration tool for documents residing on SMB or Google Drive storage. Migration Manager checks and fixes documents prior to upload then batches migrations to Azure and then to SharePoint Online. Migration Manager supports migration to SharePoint Online or OneDrive for Business
