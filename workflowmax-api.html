WorkflowMax Module

The WorkflowMax Module for Windows PowerShell allows full manipulation of WorkflowMax Objects.

 

It consists of the following cmdlets

 

•Set-WFMKey - Sets the WorkflowMax Key for the current session

•Set-WFMDefaultRegion - Sets the default region for Operations

•Show-WFMKeyFetcher - Shows a UI to obtain a WorkflowMax Key

•Show-WFMKey - Shows the WorkflowMax Key for the current Session
